Overview

The control rack consists of the following elements:

  • Progress bar
    • Filmstrip
  • Control bar
    • Button controls (e.g. pause/play, 10-second rewind, etc.)
    • Additional card options (e.g. volume card, CC card, etc.)
    • Timestamp

Specs

Progress bar

  • Bar
    • Width 100%, height 4px
    • Color: #676767
    • Apply box-shadow for a softer look:inset 0 -5px 10px rgba(0, 0, 0, 0.1)
  • Progressed
    • Color: custom color
  • Scrubber
    • Circle
    • Width 8px, height 8px
    • Color: #fff
  • Ad marker
    • Width 2px, height 6px
    • Color: #fff
    • Opacity: 60%

Filmstrip

  • Container
    • Width 150px, height to fit content
    • Padding: 5px
    • Color: #000
    • Opacity: 70%
    • Border radius: 3px
    • Arrow
      • Width 20px, height 10px
      • Centered to the container
  • Thumbnail
    • Width 140px, height to fit the ratio
  • Timestamp
    • Font: 11px, #fff
    • Center align
    • Padding top: 5px

Control Bar & Button Controls

  • Control bar
    • Width 100%, height 35px, excluding the progress bar
    • Background color: #000
    • Opacity: 50%
    • Padding: 5px on both left & right, 1px on top
  • Button controls
    • Width auto, height 16px
      • Next episode button is smaller than the others, which is currently controlled with the padding of the .svg/font itself
    • Padding: 10px on both left & right
    • Color: #fff
    • Opacity: 80%

Timestamp & Additional card option

  • Additional card: Volume

Mobile

On mobile, all the controls are hidden except for the play/pause button at the center of the player.

Play/pause button

  • Width 50px, height 50px
  • Border: 2px solid
  • Triangle icon: Width auto, height 16px. Centered within the circle
  • On hover
    • Scale 150%

Skip Intro Button

This button is an optional feature that can be turned on or off. It has one single state and after being clicked, the button will fade to the right and the video will skip the introduction and jump to the beginning of a video.

Skip Intro Button: Single State

  • Width: 150 px, Height: 36 px
  • Padding: 9×35 px
  • Located on the bottom right corner of the player

  • Font family: Open Sans
  • Font size: 16 px
  • Font weight: 700
  • Font color: #D1D7DD

  • Background color: rgba(0, 0, 0, 0.4)

  • Border radius: 4 px
  • Border weight: 1 px solid
  • Border color: #D1D7DD
